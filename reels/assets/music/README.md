# Reel background music (optional)

Drop **royalty-free** `.mp3` or `.m4a` tracks in this folder and the reel
composer will pick one at random per reel.

- If this folder is empty, reels render **silent** (still valid to post).
- Use only music you are licensed to use. Licensed/commercial songs get
  auto-flagged on Instagram/Facebook and can mute or block the reel.
- Good free sources: YouTube Audio Library, Pixabay Music, Uppbeat (with the
  free-tier credit), Mixkit.

Keep clips short (the composer trims to the reel length).
